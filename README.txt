# نظام إدارة المركبات العسكرية (نسخة مقسمة)
- هذه النسخة تُقسم الكود إلى ملفات مستقلة: `index.html`, `styles.css`, `app.js`, و `sw.js`.
- جميع الميزات الأصلية موجودة: الحفظ المحلي، الاستيراد/التصدير (Excel/CSV/JSON)، البطاقات + QR، لوحة معلومات، التصفية، الفرز، Virtual Rows، أدوار المستخدمين (Viewer/Editor/Admin) مع PIN، تراجع/إعادة، تحديد متعدد، نسخ احتياطي/استرجاع، تقرير PDF، طباعة، وثيمات، وزر تعبئة بيانات.
- تمت ترقية الـ PWA إلى Service Worker حقيقي مع `sw.js`.

## تشغيل محلي
1) افتح `index.html` مباشرةً في المتصفح لتجربة الأساسيات.
2) من الأفضل تشغيل خادم محلي (مثل إضافة **Live Server** في VS Code) لتفعيل PWA بالكامل.
